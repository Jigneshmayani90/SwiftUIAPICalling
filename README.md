## Requirements

pod 'RxCocoa'

pod 'RxSwift'

pod 'Alamofire'

pod 'NVActivityIndicatorView'

## Installation

#### Manually
1. Download the project.
2. Add necessary files in your project.
3. Congratulations!  

## Usage example
To run the example project, clone the repo, and run pod install from the Example directory first.
